# squirrellm

Lightweight Openrouter Chat Client. 

> **Work in Progress. Expect errors.**

## Run
```bash
cp .env.example .env
cp docker-compose.yml.example docker-compose.yml
docker compose up -d
```
> Do not forget to set your .env file

## Development
```bash
cp .env.example .env
bun install
bun dev
```
